javascript介绍

  1. JavaScript是互联网上最流行的脚本语言，可用于Web、HTML，广泛用于服务器、pc端、移动端
  2. 特点：
    • JavaScript是一种轻量级的编程
    • JavaScript是可以插入HTML页面的编程代码
    • JavaScript插入HTML页面后，可由所有浏览器执行
    • 不限制使用次数
  3. 语句：
    • JavaScript语句是向浏览器发出命令，告诉浏览器该做什么
    • 语句间用英文分号";"，可选的
    • 按照编写顺序执行的，如果同一选择器，后面执行覆盖前面
    • 标识符必须以字母、下划线、美元符号开始
    • JavaScript关键词，不能当作标识符
    • 对大小写敏感
    • 空格不影响，不能在单词中间换行
    • 保留字，不建议使用做标识符
    • 单行注释// 多行注释/*...*/
  4. 快捷键：
    1. ctrl+/ 能快速注释html、css、js代码

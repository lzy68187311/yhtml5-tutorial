1.Command line

  • node

    1. node -v
    2. node //进入命令交互模式
    3. sudo npm install -g n //安装版本管理
    4. sudo n stable //安装最新稳定版的node

  • npm

    1. npm config ls //查看 npm config
    2. npm config ls -l
    3. npm -v //显示版本，检查npm 是否正确安装。
    4. npm ls //列出已安装模块
    5. npm ls -g
    6. npm install express //安装express模块
    7. npm install -g express //全局安装express模块
    8. npm uninstall express //删除指定的模块
    9. npm update //升级当前目录下的项目的所有模块
    10. npm update express
    11. npm update -g express //升级全局安装的express模块
    12. npm show express //显示模块详情
    13. npm search express
    14. npm publish
    15. npm cache clean
    16. npm config set proxy="http://127.0.0.1:8087" //设置代理地址 http://www.xxxx.com:port格式
    17. npm config set registry="http://registry.npmjs.org" //设置npm源地址
    18. npm install express --registry=http://registry.npmjs.org //e.g 安装时临时修改npm源地址,成功

  • fis3

    1. fis3 -v
    2. fis3 release -d ./output //构建发布到项目目录的 output 目录下
    3. fis3 release -d ../dist //构建发布到项目父级目录的 dist 子目录下
    4. 进入项目根目录(配置文件fis-conf.js所在的目录)，执行命令，进行构建。

2.Issue

  1. permission denied 权限问题//sudo chmod -R 777 /usr/local/
  2. 安装不成功，可能被墙，使用淘宝的cnpm

3.DOS 常用命令

  1. cd 目录名：进入特定的目录。如果看到有个目录显示为：abcdef ghi 就 输入：cd abcdef.ghi进入该目录。
  2. cd\ 退回到根目录。
  3. cd..退回到上一级目录
  4. cls 清除屏幕
